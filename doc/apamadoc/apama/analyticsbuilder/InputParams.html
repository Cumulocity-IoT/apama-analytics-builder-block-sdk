apama.analyticsbuilder
Event InputParams

Parameters for describing a stream of events consumed by an input block.

The combination of event type, field values, and partition value identifies the input stream.

This is typically used as: 

InputHandler inputHandler;

action $validate() {
	InputParams params := InputParams.forEventType(MyEvent.getName(), new dictionary).withPartitionValue(source);
	inputHandler := $base.consumesInput(params);
}
See Also:
apama.analyticsbuilder.BlockBase#consumesInput() - To declare an input block.
