Action detail

schedule

apama.analyticsbuilder.TimerHandle schedule(any inputEvent, optional<float> timestamp, any partition)
Schedule the input event to be processed at the specified time. If no time is specified then the input event is scheduled to be processed as soon as possible.

This creates a timer for the specified timestamp or the earliest possible time. When the timer is triggered, the $timerTriggered action of the block is called and the inputEvent is passed as the payload.
Parameters:
inputEvent - The input event.
timestamp - The source timestamp of the event. Pass empty value if the source timestamp is not to be used or not available.
partition - The partition to which the input value belongs, if it is partitioned. Pass an empty value if the input is not partitioned.
Returns:
The handle to the timer created.

scheduleNow

apama.analyticsbuilder.TimerHandle scheduleNow(any value, any partition)
Schedule the input event to be processed as soon as possible.

This creates a timer for the earliest possible time. When the timer is triggered, the $timerTriggered action of the block is called and the value is passed as payload.
Parameters:
value - The input event.
partition - The partition to which the input value belongs, if it is partitioned. Pass empty value if the input is not partitioned.
Returns:
The handle to the timer created.
