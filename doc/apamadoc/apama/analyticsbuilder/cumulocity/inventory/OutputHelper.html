apama.analyticsbuilder.cumulocity.inventory
Event OutputHelper

Deprecated:
This helper event has been deprecated, use the CumulocityOutputParams and CumulocityOutputHandler events instead.
Helper methods for output blocks.

Helpers for output blocks to process InventoryLookupResults, calling sendsSyncOutput. This is typically used as so: 

OutputHelper helper := OutputHelper.forBlock(self, $modelScopeParameters);
helper.setSyncOutput($parameters.deviceId, Measurement.getName(), {"type":$parameters.measurementType, "fragment":fragment, "series":series});

return InventoryLookup.lookup($parameters.deviceId).andThen(helper.declareOutput);
(with other callbacks before helper.declareOutput if needed).
See Also:
CumulocityOutputParams - To declare an output block.
CumulocityInputHandler - To send an output event.

declareOutput

any declareOutput(any a_lookupResult)
Promise callback for declaring output.

This will call BlockBase.sendsSyncOutput or BlockBase.sendsAsyncOutput. If the block has deviceId or devices members, they are set from the InventoryLookupResult.
Parameters:
a_lookupResult - The promised result, which should be an InventoryLookupResult.
