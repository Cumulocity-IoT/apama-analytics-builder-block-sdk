absolute

apama.analyticsbuilder.TimerParams static absolute(float timeSec)
Create a TimerParams for creating a timer for an absolute time.

The maximum time delay possible for input events is taken into account when determining the actual trigger time.
Parameters:
timeSec - The absolute time in seconds at which the timer should be triggered.
Returns:
A new TimerParams for creating a timer for an absolute time.
