apama.analyticsbuilder
Event PromiseReturner

Helper class for handling promises.

A promiseReturner ignores the promise's previous value and returns (or throws) a new value. Usage: 
 promise.andThen(PromiseReturner.create(object).andReturn)
This will wait for promise to complete, then ignore it's return value and return object.
