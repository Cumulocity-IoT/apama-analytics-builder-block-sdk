sendOutput

any sendOutput(any outputEvent, string channel, apama.analyticsbuilder.Activation $activation)
Send the output event outside of the model to the specified channel.

If the output is synchronous then it is routed internally for the consumption by other models and tagged before sending it outside.
Parameters:
outputEvent - The output event to send.
channel - The channel on which to send the output event.
$activation - The current activation.
Routes:
any - The output event if it is synchronous.
Sends:
any - The output event to the output channel.
